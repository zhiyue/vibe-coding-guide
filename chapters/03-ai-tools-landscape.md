# 第三章：AI 编程工具全景

> **本章你将学到：** 市面上有哪些 AI 编程工具，它们各自适合什么场景，以及作为新手应该怎么选。

## 别被选择吓到

如果你去搜索"AI 编程工具"，会发现结果多到让人头晕——Bolt、Cursor、Copilot、Claude Code、v0、Lovable、Trae、Codex……名字一个比一个酷，但到底是什么？有什么区别？

别慌。我帮你把它们分成三大类，搞清楚分类之后，选择就简单多了。

本章的目标不是让你记住每个工具的所有细节，而是让你**了解有哪些选择**，为下一章选定一个工具做好准备。

## 三大类：一个简单的分类

AI 编程工具虽然多，但按照使用方式，基本可以分为三类：

| 类型 | 一句话解释 | 类比 |
|------|----------|------|
| **网页端工具** | 打开浏览器就能用 | 像点外卖——打开 App，说你要什么，等着送来 |
| **编辑器端工具** | 需要在电脑上安装一个软件 | 像在家做饭——厨房（工具）更全，能做的菜更多 |
| **命令行端工具** | 在终端里用文字命令操作 | 像专业厨房——功能最强，但需要知道怎么操作设备 |

接下来我分别介绍每一类里的主要工具。

---

## 网页端工具：打开浏览器就能用

这类工具最大的优点是**零安装、即开即用**。你只需要一个浏览器和一个账号，打开网站就可以开始。特别适合刚接触 vibe coding 的新手。

### Bolt.new

**一句话定位：** 目前最主流的网页端 AI 编程工具之一，全栈能力强，适合做完整的网站和应用。

你在 Bolt.new 的输入框里描述你想要的东西，它就会直接帮你生成一个完整的项目——包括前端界面、后端逻辑、甚至数据库。你可以在右侧实时预览效果，不满意就继续对话修改。

<!-- screenshot: Bolt.new 的主界面截图，标注输入框、预览区域、文件区域 -->

**适合做什么：** 个人网站、小工具、完整的 Web 应用原型

**优点：**
- 功能全面，前后端都能做
- 预览效果直观
- 可以一键部署上线

**缺点：**
- 免费额度用完后需要付费
- 复杂项目可能需要较多轮对话

**价格：** 免费额度每月约 100 万 token（够做好几个小项目），付费版 $25/月

### v0.dev

**一句话定位：** Vercel 出品，特别擅长做好看的 UI 界面和前端组件。

如果你特别在意页面的视觉效果，v0 是个不错的选择。它生成的界面设计感通常比较好，特别适合做需要"好看"的页面。

<!-- screenshot: v0.dev 的界面截图 -->

**适合做什么：** 着重视觉效果的页面、UI 组件、前端原型

**优点：**
- 生成的 UI 设计感好
- 支持 Figma 设计稿导入
- 可以直接部署到 Vercel

**缺点：**
- 更偏重前端，后端功能相对弱
- 免费额度较少

**价格：** 免费每月 $5 额度，付费版 $20/月

### Lovable

**一句话定位：** 操作体验友好的全栈 AI 编程工具，适合快速把想法变成可用的应用。

Lovable 的交互设计比较直观，也可以直接在浏览器里完成从创建到部署的全流程。

<!-- screenshot: Lovable 的界面截图 -->

**适合做什么：** 全栈应用、快速原型

**优点：**
- 界面友好，操作直觉化
- 支持自定义域名
- 内置部署功能

**缺点：**
- 免费额度有限（每天 5 次）
- 品牌标识需要付费才能移除

**价格：** 免费每天 5 次使用，付费版 $25/月

### Replit Agent

**一句话定位：** 在线编程平台 Replit 的 AI 功能，从创建到运行到部署都在浏览器里完成。

Replit 本身就是一个成熟的在线编程平台，它的 AI Agent 功能让你可以用自然语言指挥它写代码。

<!-- screenshot: Replit Agent 的界面截图 -->

**适合做什么：** 各种类型的项目，包括后端服务

**优点：**
- 平台成熟稳定
- 支持多种编程语言
- 内置运行环境，不需要额外配置

**缺点：**
- AI Agent 功能需要付费
- 界面对新手来说信息量偏大

**价格：** 基础功能免费，AI Agent 需要 Replit Core 订阅（$25/月）

## 编辑器端工具：功能更强大的选择

这类工具需要在电脑上安装一个专门的软件（编辑器）。虽然多了安装这一步，但功能会比网页端更全面，适合做更复杂的项目。

### Cursor

**一句话定位：** 目前最主流、最受欢迎的 AI 编程编辑器，在 VS Code 基础上深度集成了 AI 能力。

Cursor 是基于 VS Code（微软出的代码编辑器）改造的。如果说网页端工具是"点外卖"，那 Cursor 就是给你一个功能齐全的厨房，而且还配了一个 AI 厨师。你可以在编辑器里直接和 AI 对话（通过 Composer 功能），AI 会直接在你的代码里做修改。

<!-- screenshot: Cursor 的界面截图，标注 Composer 对话面板、代码编辑区域、预览区域 -->

**适合做什么：** 几乎所有类型的项目，从简单页面到复杂应用

**优点：**
- 功能最全面，是目前 AI 编程的标杆产品
- 基于 VS Code，插件生态丰富
- AI 直接在代码里修改，而不只是生成新文件
- 支持多种 AI 模型（Claude、GPT 等）

**缺点：**
- 需要安装，有一点入门门槛
- 免费额度有限
- 对电脑性能有一定要求

**价格：** 免费版有限额度 + 7 天 Pro 试用，Pro 版 $20/月。学生可免费用一年 Pro。

### Trae

**一句话定位：** 字节跳动推出的 AI 编程编辑器，同样基于 VS Code，免费额度慷慨，**对中文用户特别友好**。

Trae 是 Cursor 的有力竞争对手。它的最大卖点是：免费额度非常慷慨，而且**原生支持中文**——界面是中文的，跟 AI 对话用中文效果也很好。对于中文用户来说，这是一个非常实际的优势。

它有一个独特的 **Builder 模式**，会在动手写代码之前先"想一想"，规划好方案再执行，结果的准确性通常不错。

<!-- screenshot: Trae 的中文界面截图，标注 Builder 模式 -->

**适合做什么：** 各种类型的项目，尤其适合中文用户

**优点：**
- 免费额度慷慨（每月 5000 次代码补全 + AI 对话额度）
- 原生中文支持，体验最好
- Pro 版只要 $10/月，是 Cursor 的一半
- 支持多种高端 AI 模型（Claude、GPT-4o、Gemini）
- 支持支付宝付费，对国内用户友好

**缺点：**
- 相比 Cursor 社区和插件生态稍小
- 作为较新产品，稳定性可能略逊
- 存在数据隐私方面的讨论

**价格：** 免费额度慷慨，Pro 版 $10/月（首月 $3），年付 $90。

### Windsurf

**一句话定位：** 另一款 AI 编程编辑器，特点是支持自带 API Key 使用。

Windsurf（前身叫 Codeium）也是基于 VS Code 的 AI 编辑器。它的独特之处在于支持"自带钥匙"（BYOK）——你可以接入自己的 AI 模型 API，不受工具本身的额度限制。

<!-- screenshot: Windsurf 的界面截图 -->

**适合做什么：** 各种类型的项目

**优点：**
- 支持自带 API Key，灵活度高
- Pro 版 $15/月，比 Cursor 便宜
- 内置部署功能

**缺点：**
- 免费额度比较少（每月 25 次）
- "自带 API Key"这个优势对新手意义不大

**价格：** 免费每月 25 次使用，Pro 版 $15/月

### GitHub Copilot

**一句话定位：** GitHub 和微软出品的 AI 编程助手，主要以代码补全和对话辅助为主。

Copilot 是最早火起来的 AI 编程工具之一。它不是一个独立的编辑器，而是一个"插件"，可以装在 VS Code、Cursor 等编辑器里面。它最擅长的是在你写代码的时候自动补全——你打了一半的句子，它帮你补完。

<!-- screenshot: GitHub Copilot 在 VS Code 中的代码补全效果 -->

**适合做什么：** 辅助写代码，代码补全，对话式编程

**优点：**
- 免费版就有 2000 次/月的代码补全
- 支持多种编辑器
- 微软和 GitHub 背书，稳定可靠
- 学生和开源维护者免费

**缺点：**
- 不是完整的 AI 编程解决方案，更多是辅助角色
- 单独使用不如 Cursor 等专门的 AI 编辑器体验好

**价格：** 免费版 2000 次补全/月 + 50 次 AI 对话，Pro 版 $10/月

### Qoder

**一句话定位：** 阿里巴巴推出的 AI 编程 IDE，亮点是 Quest 模式——描述需求后 AI 全自动完成编码。

Qoder 是 2025 年下半年才推出的新工具。它最独特的功能是 **Quest 模式**：你只需要描述你想做什么，Qoder 就会完全自主地去编码、测试、交付，中间不需要你干预。有点像你给 AI 下了一个"任务单"，它自己去完成。

<!-- screenshot: Qoder 的 Quest 模式界面截图 -->

**适合做什么：** 全自动编码任务，适合想要"全程托管"体验的用户

**优点：**
- Quest 模式真正的全自动，对新手很友好
- 目前处于公测阶段，完全免费
- 支持多种 AI 模型（Claude、Gemini、GPT）

**缺点：**
- 还在公测，不够成熟
- 长期定价未知
- 社区和生态比 Cursor 小很多

**价格：** 目前公测期间免费

## 命令行端工具：给进阶用户的利器

这类工具在"终端"（一个黑色的窗口，用文字命令操作电脑）里运行。功能最灵活，但对新手来说入门门槛最高。

**作为 vibe coding 新手，你现在不需要学这类工具。** 但了解一下它们的存在，等你将来有兴趣了可以尝试。

### Claude Code

**一句话定位：** Anthropic 公司出品的终端编程 Agent，直接在命令行里和 AI 对话来写代码。

Claude Code 是一个在终端里运行的 AI 编程工具。你在命令行里用自然语言告诉它你要做什么，它就会在你的项目文件夹里直接读取、修改、运行代码。它可以理解整个项目的上下文，做出跨文件的复杂修改。

**适合做什么：** 复杂项目的开发、代码重构、需要理解整个代码库的任务

**优点：**
- AI 能力强（基于 Claude 模型）
- 能理解整个项目结构
- 适合做复杂的跨文件修改

**缺点：**
- 需要终端基础，新手不友好
- 需要付费订阅才能使用（最低 $20/月）
- 国内用户无法直接使用官方 API（后面会讲替代方案）

**价格：** 需要 Claude Pro 订阅 $20/月起

#### 国内用户使用 Claude Code 的替代方案

由于网络原因，国内用户无法直接使用 Anthropic 的官方 API。但好消息是，国内有多个平台提供了兼容的替代方案，让你可以用国产 AI 模型来驱动 Claude Code 等工具：

| 平台 | 入门价格 | 背后模型 | 特点 |
|------|---------|---------|------|
| **火山引擎 Coding Plan** | 8.9 元/首月 | 豆包、GLM、DeepSeek、Kimi 四选 | 模型选择最多 |
| **Kimi Code** | 39 元/月 | Kimi K2.5 | 速度最快（100 token/秒），有独立 CLI |
| **MiniMax Coding Plan** | $10/月 | MiniMax M2.5 | 配额最透明，内置搜索和图片理解 |
| **智谱 GLM Coding** | ~20 元/月 | GLM-4.7 / GLM-5 | 国产模型中编程能力最强 |

这些平台都兼容 Claude Code 的接口协议——简单来说，你只需要改一个配置地址，Claude Code 就能用国产模型来工作。具体配置方法各平台官网都有教程。

- 火山引擎：https://www.volcengine.com/activity/codingplan
- Kimi Code：https://www.kimi.com/code
- MiniMax：https://platform.minimaxi.com/docs/pricing/coding-plan
- 智谱 GLM Coding：https://www.bigmodel.cn/glm-coding

### OpenAI Codex CLI

**一句话定位：** OpenAI 官方开源的终端编程 Agent，使用 GPT 模型在命令行中编程。

Codex CLI 是 OpenAI 推出的开源终端编程工具，功能和 Claude Code 类似。它使用 GPT-5 模型，同样可以在命令行里读取、修改和运行代码。

**适合做什么：** 终端编程，需要 GPT 模型的场景

**优点：**
- 开源，社区活跃（GitHub 6 万+ Stars）
- 基于 GPT-5 模型
- 可以配合 ChatGPT Plus 订阅使用

**缺点：**
- 需要 ChatGPT 付费订阅（$20/月起）
- 终端操作对新手不友好
- 国内访问不便

**价格：** 需要 ChatGPT Plus $20/月，或使用 API 按用量付费

### Aider

**一句话定位：** 开源的终端编程工具，可以搭配多种 AI 模型使用。

Aider 是一个开源项目，特点是可以灵活搭配不同的 AI 模型（Claude、GPT、本地模型等），并且与 Git 版本管理深度集成。

**适合做什么：** 灵活搭配 AI 模型的终端编程

**优点：** 开源免费、模型选择灵活、Git 集成好
**缺点：** 需要终端基础、需要自备 AI 模型的 API Key
**价格：** 工具本身免费，需要自付 AI 模型费用

## 新手应该选哪个？

看完这么多工具，是不是反而更迷茫了？没关系，我帮你简化：

### 推荐路径

**如果你是完全的新手，想最快体验 vibe coding：**
→ 先用 **Bolt.new**（打开浏览器就能用，不需要装东西）

**如果你是国内用户，想要更好的中文体验和更实惠的价格：**
→ 用 **Trae**（免费额度慷慨，原生中文支持，首月 Pro 只要 $3）

**如果你体验了一段时间，想要更强大的工具：**
→ 用 **Cursor**（目前公认最好的 AI 编程编辑器）

**如果你想尝鲜免费的全自动编程：**
→ 试试 **Qoder**（公测免费，Quest 模式很新奇）

### 最重要的一句话

**不要纠结选哪个——用起来比选对更重要。**

所有这些工具都在快速迭代，今天的差距明天可能就被填平了。选一个先开始，如果觉得不合适，随时可以换。你在一个工具上学到的技巧（尤其是怎么写好 prompt），在其他工具上一样适用。

## 本章小结

- AI 编程工具分三类：网页端（最简单）、编辑器端（功能更全）、命令行端（最灵活但门槛高）
- 网页端代表：Bolt.new、v0.dev、Lovable
- 编辑器端代表：Cursor（最主流）、Trae（中文友好、便宜）、Windsurf、Copilot、Qoder
- 命令行端代表：Claude Code、Codex CLI、Aider
- 国内用户可通过火山引擎、Kimi Code、MiniMax、智谱等平台使用 Claude Code
- 新手推荐：先用 Bolt.new 体验，再转到 Trae 或 Cursor
