# 第二章：你需要准备什么

> **本章你将学到：** 开始 vibe coding 之前需要准备的东西——硬件、账号和心态。

## 好消息：你需要的比你想的少

很多人一听到"编程"两个字就觉得门槛很高——是不是要买很贵的电脑？是不是要装一堆看不懂的软件？是不是要先学半年才能开始？

都不需要。

Vibe coding 的入门准备清单非常短。这一章我帮你把需要准备的东西理清楚，让你可以尽快开始。

## 心态准备：最重要的事

在聊具体要准备什么之前，我想先说说心态——因为这才是最重要的。

**你不是在"学编程"，你是在"学和 AI 沟通"。**

这个区别非常关键。传统的学编程，你需要记住一大堆语法规则、理解复杂的逻辑概念，那确实需要花很长时间。但 vibe coding 不一样——你要学的核心技能是：**怎么把你的想法清楚地告诉 AI**。

这就像学会使用搜索引擎一样。二十年前，很多人也不会用百度或 Google——不知道该搜什么关键词，不知道怎么筛选结果。但现在，搜索已经是大家日常生活的一部分了。Vibe coding 也一样，它的核心就是学会"提问"。

另一个重要的心态是：**不需要追求完美，只需要不断迭代**。你的第一个作品不会是完美的，但没关系。做出来，看看效果，哪里不满意就告诉 AI 去改。改一版，再改一版，慢慢就越来越好了。就像写文章一样，初稿不可能完美，重要的是先写出来，然后不断修改。

## 硬件要求

**一台能上网的电脑。** Windows、Mac 或者 Linux 都行，不需要很新或者很贵。只要能流畅地打开网页，就够了。

**一个现代浏览器。** 推荐用 Chrome（谷歌浏览器），因为大部分 AI 编程工具对 Chrome 的兼容性最好。当然，Edge、Firefox 也可以。

就这么多。真的。

你不需要高配电脑，不需要显卡，不需要特别大的硬盘空间。因为 vibe coding 的大部分工作是 AI 在云端完成的，你的电脑主要就是用来跟 AI 对话和预览结果。

## 需要注册的账号

开始之前，你需要注册几个账号。别担心，都很简单。

### AI 编程工具账号

这是你最核心的工具。市面上有很多选择，比如 Bolt.new、v0.dev、Cursor 等等。每个工具具体是什么、怎么选，我们在第三章和第四章会详细介绍。

这里你只需要知道：**你至少需要注册一个 AI 编程工具的账号**。具体注册哪个，等看完第三章再决定也不迟。

### GitHub 账号

**GitHub 是什么？** 你可以把它理解为程序员的"云端硬盘"。就像你用百度网盘存文档和照片一样，程序员用 GitHub 来存代码。

**为什么需要？** 两个原因：
1. 很多 AI 编程工具需要用 GitHub 账号登录
2. 以后你可以把自己做的项目存到 GitHub 上，方便管理和分享

**怎么注册？**
1. 打开 https://github.com
2. 点击右上角的 "Sign up"
3. 输入邮箱、设置密码、选一个用户名
4. 按提示完成验证就行了

## 认识终端/命令行

你可能听过"命令行"或"终端"这两个词，听起来像是黑客才用的东西。但其实它非常简单——**终端就是用文字和电脑对话的地方**。

平时你操作电脑是用鼠标点点点，而终端是用打字的方式告诉电脑要做什么。就像发微信一样——你打字发一条指令，电脑回复你结果。

为什么要知道这个？因为后面安装一些工具时，我们偶尔需要在终端里输入一行命令。别担心，每次用到的时候我都会告诉你具体输入什么，你只需要复制粘贴就行。

### 怎么打开终端？

**Mac 用户：**
1. 按 `Command + 空格`，打开"聚焦搜索"
2. 输入"终端"或"Terminal"
3. 回车打开

<!-- screenshot: Mac 打开终端的方式，聚焦搜索输入"终端" -->

**Windows 用户：**
1. 按 `Win + X`，选择"Windows PowerShell"或"终端"
2. 或者在开始菜单搜索"PowerShell"

<!-- screenshot: Windows 打开 PowerShell 的方式 -->

### 试一试

打开终端后，输入下面这行字，然后按回车：

```
echo "Hello, Vibe Coding!"
```

如果屏幕上显示了 `Hello, Vibe Coding!`，恭喜你，你已经成功使用终端了。就这么简单。

<!-- screenshot: 终端中运行 echo 命令的效果 -->

后面的章节里，遇到需要在终端操作的步骤，我都会用这样的代码框告诉你具体输入什么。你只需要打开终端、复制粘贴、按回车就行。

## 按路线准备环境

> **不确定选哪条路线？** 没关系，先跳过这一节。第三章和第四章会帮你选工具，选好后再回来按需安装。

> **写给第一次配置环境的你：**
>
> 如果你是第一次接触这些东西，遇到问题是**完全正常的**——即使是工作多年的程序员，配环境时也经常要搜索解决方案。几点建议：
>
> - **不要气馁：** 安装出问题不代表你不适合做这个，每个人都经历过
> - **仔细看错误信息：** 屏幕上显示的红字通常包含解决问题的线索
> - **善用搜索：** 把错误信息复制到搜索引擎，大概率能找到解决方案
> - **让 AI 帮你：** 这正是 vibe coding 的精髓——把错误信息、截图或问题描述发给 ChatGPT、Claude 等 AI 工具，让它帮你分析原因并给出解决步骤
> - **一步一步来：** 按顺序执行，每一步都验证成功后再进行下一步
> - **只需要做一次：** 环境配好之后就不用再折腾了，之后就可以专注于创作

不同的工具需要不同的准备工作。下面按路线来说明：

### 路线 A：网页端工具（Bolt / v0 / Lovable）

**恭喜，你什么都不用装！**

这类工具全部在浏览器里运行，打开网页就能用。你可以直接跳到下一章了。

### 路线 B：编辑器端工具（Cursor / Trae）

如果你想用 Cursor 或 Trae 这类需要安装在电脑上的工具，需要多准备几样东西。按顺序来就行，每一步都很简单。

#### 1. 安装 Homebrew（仅 Mac 用户需要）

Homebrew 是 Mac 上的"软件应用商店"——通过它可以用一行命令安装各种开发工具，比你到处找下载链接方便多了。

打开终端，粘贴下面这行命令，按回车（这条命令看着很长很复杂，但你不需要理解它——直接复制粘贴就行）：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

安装过程中可能会让你输入电脑的开机密码（输入时屏幕上不会显示任何字符，这是正常的），输入后按回车就行。

整个过程可能需要几分钟，耐心等待即可。

<!-- screenshot: 终端中 Homebrew 安装完成的提示 -->

**验证安装成功：** 输入以下命令：

```bash
brew --version
```

如果看到类似 `Homebrew 4.x.x` 的版本号，就说明成功了。

> **安装失败或很慢？** 国内用户可以使用镜像加速。在终端中执行：
> ```bash
> /bin/bash -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
> ```
> 按提示选择镜像源即可。

**Windows 用户：** 你不需要 Homebrew。Windows 上安装软件直接从官网下载安装包就行，后面会给你具体的下载链接。

#### 2. 安装 Node.js

Node.js 是让 JavaScript（网页常用的编程语言）能在你电脑上运行的工具。很多 AI 编程工具生成的项目都需要它。

**Mac 用户（已装 Homebrew）：**

```bash
brew install node
```

**Windows 用户：**
1. 打开 https://nodejs.org
2. 下载标有 "LTS"（长期支持版）的版本
3. 双击安装，一路"下一步"

<!-- screenshot: Node.js 官网下载页面，标注 LTS 版本的下载按钮 -->

**验证安装成功：**

```bash
node --version
```

看到类似 `v22.x.x` 的版本号就说明成功了。

#### 3. 安装 Cursor 或 Trae

- **Cursor：** 从 https://cursor.com 下载安装
- **Trae：** 从 https://trae.ai 下载安装

安装方式和普通软件一样，双击安装包，按提示操作即可。

<!-- screenshot: Cursor 安装完成后首次打开的界面 -->

### 可选：安装 Python

如果你将来想做非网页类的项目（比如数据处理、文件批量操作、自动化脚本），可以顺便把 Python 也装上。不确定要不要装？先跳过，以后需要时再回来。

**Mac 用户：**

```bash
brew install python
```

**Windows 用户：**
1. 打开 https://python.org
2. 下载最新稳定版
3. 安装时**一定要勾选 "Add Python to PATH"**（很重要！）

<!-- screenshot: Python 安装界面，标注"Add to PATH"的勾选框 -->

**验证安装成功：**

```bash
python3 --version
```

看到类似 `Python 3.12.x` 的版本号就说明成功了。Windows 用户可能需要用 `python --version`（没有 `3`）。

## 本章小结

- 开始 vibe coding 只需要：一台能上网的电脑 + 一个浏览器
- 心态上最重要的是：你在学"和 AI 沟通"，不是在"学编程"
- 注册一个 GitHub 账号（免费）
- 所有 AI 编程工具都有免费额度，不花钱就能开始
- 终端不可怕——就是用文字和电脑对话，需要时复制粘贴命令就行
- 用网页端工具（Bolt/v0）不需要任何环境配置
- 用编辑器端工具（Cursor/Trae）需要安装 Node.js，Mac 用户建议先装 Homebrew
